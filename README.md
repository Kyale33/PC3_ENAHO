# Análisis de la informalidad utilizando datos de la ENAHO

## Descripción del proyecto
Este repositorio incluye el código y el flujo de trabajo completo del "Informe sobre la situación de la informalidad en el Perú de 2021-205" para el curso del Taller de Precesamiento de Datos 2025-1.
Se utilizan datos de la Encuesta Nacional de Hogares (ENAHO) del 2021-2025 trabajados integramente en R

El análisis explora la relación entre la informalidad y las siguientes dimensiones:
**Demográficas**: Ingreso, edad, nivel educativo, lengua materna y ubicación geográfica
**Institucionales y de seguridad social**: Confianza en las instituciones, salud (acceso, aseguramiento, gasto de bolsillo) y pensiones

## Librerías utilizadas
El proyecto está desarrollado utilizando la versión 4.5.3 de RStudio, con las siguientes librerías:
- *tidyverse: para el procesamiento y gráficos (dplyr y ggplot)
- rio
- arrow
- renv
- janitor
